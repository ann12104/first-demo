# 猫咪钓鱼陪伴游戏 \- 1页极简策划案（独立开发版）

## 一、核心定位（核心灵魂）

类型：2D治愈陪伴类（2\.5D伪3D体积感）\+ 轻度钓鱼\+收集养成
核心氛围：安静、温柔、无压力、陪伴感（拒绝肝氪、拒绝广告骚扰）
目标平台：仅Google Play（Android）
包体目标：≤50MB（2D项目\+资源压缩）
目标地区：印尼、泰国、越南（东南亚治愈向核心用户）
核心卖点：钓鱼时猫咪全程陪伴，主动叼礼物，共同布置温馨猫舍

## 二、核心玩法闭环（自洽无冗余）

1. **钓鱼（核心行为）**：一键抛竿、自动上钩（3\-5秒/次），界面纯净无广告；产出「基础物资」（石子、水草等）\+「食物」（小鱼干等），无失败惩罚，纯放松体验。

2. **猫咪互动（情感核心）**：钓鱼时猫咪全程在旁（趴卧、蹭手、玩耍）；消耗「食物」喂猫，或陪伴钓鱼累计时长，可提升「好感度」，同时降低猫咪「疲惫值」。

3. **猫咪送礼（惊喜反馈）**：「好感度」越高，猫咪主动叼礼物的概率越高，礼物品质越好（普通：小花/树叶 → 优质：羽毛/贝壳 → 稀有：小宝石/蝴蝶）；送礼无规律，主打随机惊喜感。为避免“脸黑”影响体验，加入轻保底：连续8次钓鱼未触发送礼时，第9次钓鱼触发一次普通及以上礼物。

4. **猫舍布置（长期目标）**：猫咪叼来的礼物放入猫舍展示架，不同品质礼物提供不同「温馨值（舒适度）」；温馨值越高，猫咪「疲惫值恢复速度越快」，可解锁新展示架、新猫舍场景、新猫咪外观。

5. **物资兑换（补充玩法）**：钓鱼产出的「基础物资」，可兑换猫咪装扮、猫窝皮肤、场景装饰，进一步提升「温馨值」，无复杂兑换逻辑。

## 三、核心数值体系（轻量、情感化，无复杂计算）

|数值名称|作用|提升/恢复方式|消耗/降低方式|
|---|---|---|---|
|好感度|提升猫咪送礼概率、提升礼物品质|喂食物、陪伴钓鱼（累计时长）、看激励广告（自愿）|首版不做自然回落，避免压力感|
|疲惫值|疲惫值过高，猫咪会暂时离开（不互动、不送礼）|猫舍温馨值越高，恢复越快；喂食物可小幅恢复；看激励广告（自愿，立即恢复）|陪伴钓鱼（轻微累积）|
|温馨值|加速疲惫值恢复、解锁新内容|摆放猫咪赠送的礼物、兑换装饰|无消耗，仅随礼物摆放数量/品质提升|

## 四、猫咪行为逻辑（贴合治愈氛围，简单易实现）

- 常态行为：钓鱼时，猫咪在画面内随机趴卧、蹭手、玩耍（无复杂动画，简单帧动画即可）；

- 送礼行为：好感度达标后，随机触发（钓鱼中、钓鱼后均可），猫咪跑开后叼回礼物，自动提示放入展示架；连续8次钓鱼未触发时，第9次触发保底送礼（普通及以上品质）；

- 疲惫行为：疲惫值满后，猫咪会蜷缩休息，不再互动、不送礼，恢复后自动回到钓鱼旁；

- 互动反馈：喂食物时，猫咪会有蹭手、摇尾巴动画；收到稀有礼物时，有简单庆祝动画（不夸张、不突兀）。

## 六、广告接入方案（温和不破坏氛围，守住洁癖）

核心原则：所有广告均为玩家自愿触发，不主动弹窗、不打断游戏氛围，钓鱼界面绝对无广告

1. **激励视频（核心变现）**：4个触发点，文案温柔无吆喝
        

    - 钓鱼结束后：“看短片，猫咪多捡一份小礼物”（双倍礼物）；

    - 猫舍界面：“看短片，给猫咪加份零食，提升今日好感度”；

    - 解锁新展示架/场景：“看短片，立即为猫咪腾出新空间”；

    - 猫咪疲惫时：“看短片，让猫咪快速恢复活力”。

2. **Banner广告**：仅放在猫舍展示界面底部，样式清淡、不挡内容，钓鱼界面不出现；

3. **插屏广告**：仅在玩家点击“退出游戏”时弹出一次，不打断游玩过程。

4. **频控与兜底规则（必须实现）**：

    - 激励视频按功能做冷却（建议30\-60秒）与每日上限（建议每类6\-10次）；

    - 任意广告加载失败时，不扣玩家机会，提示“当前暂无短片，请稍后再试”并给可重试入口；

    - 关键流程触发激励失败时，可给小额安慰奖励（如普通礼物x1或疲惫值小幅恢复）避免体验断层。

## 七、开发优先级（独立开发适配，先做核心再补细节）

1. 核心玩法：钓鱼基础功能 \+ 猫咪常态陪伴动画；

2. 情感反馈：猫咪送礼逻辑 \+ 好感度/疲惫值基础数值；

3. 猫舍系统：展示架功能 \+ 温馨值计算 \+ 礼物摆放；

4. 变现接入：激励视频（核心4个点位）\+ 简单Banner；

5. 细节补充：物资兑换、猫咪外观解锁、场景优化。

6. 新手引导（极简）：仅覆盖首次启动的3步操作（点击钓鱼→喂食互动→查看展示架），全程30\-60秒，可跳过，不做强制长引导。

## 八、审核与上架注意（Google Play适配）

- 隐私政策：免费模板修改，明确“不收集个人信息，仅广告SDK可能收集广告ID”；

- 内容合规：无暴力、无低俗内容，猫咪互动、广告文案均走治愈风；

- 包体优化：2D项目（Built\-in管线），图片/音频按压缩方案处理，控制在50MB内；

- 商店描述：突出“治愈陪伴、猫咪送礼、无压力”核心卖点，贴合东南亚用户偏好。

- 每日重置与防刷：本地按自然日重置广告次数与每日奖励；若检测到系统时间异常回拨，仅允许进入游戏但暂停当日额外奖励发放。

## 九、MVP验收指标（首发判定）

- 核心闭环：单用户10分钟内可完成至少3次“钓鱼→互动→送礼/成长”循环；

- 体验稳定：广告失败不阻断主流程，关键奖励成功发放率需稳定；

- 留存目标：次日留存达到可继续投放的基础线（首批以内部目标为准）；

- 数据安全：退出重进后数据正确恢复，坏档可自动回退默认存档。

- 新手完成：首次启动引导完成率达到可接受基线，且跳过后不影响主流程。
